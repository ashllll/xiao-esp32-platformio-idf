# 硬件设计、数据手册与原理图规则

本页总结当前 ADS1115 + NTC100K B3950 工作，并声明本项目后续器件选型、数据手册和原理图绘制规则。

## 当前工作总结

本项目已经完成：

- 基于 ESP-IDF 6.0.1 新 I²C 主机 API 的 ADS1115 单次转换组件；
- NTC 低侧分压电阻反算和单 Beta 温度换算组件；
- `R25=100 kΩ`、`Beta=3950 K`、固定电阻 100 kΩ 的三板共用示例；
- 同时采集 AIN0 分压节点和 AIN1 实际激励电压，避免把 3.3 V 当成无误差常数；
- XIAO ESP32C3、ESP32S3、ESP32C6 构建验证；
- 主机单元测试、中英文接线/公式/误差预算文档及一手资料核对记录。

## 默认厂商与资料来源

当用户没有指定器件厂商时，本项目遵循以下默认规则：

1. 优先选择满足项目要求的 Texas Instruments（TI，德州仪器）器件。
2. 数据手册必须从 TI 官方 `ti.com` 获取；记录器件完整料号、文档编号、修订版本、发布日期或修订日期以及检索日期。
3. 以数据手册的 Recommended Application、Typical Application 或 TI 官方参考设计原理图作为实际项目原理图的基线。
4. 第三方模块页面、经销商页面、镜像和聚合数据手册只能用于发现线索，不能替代原厂官方资料。

如果用户明确指定了厂商或完整料号，则使用指定厂商的官方数据手册，不强行改成 TI 器件。

如果 TI 没有合适器件，或项目约束必须使用非 TI 器件：

- 不得虚构 TI 料号、错配数据手册或把其他厂商器件归到 TI 名下；
- 必须说明 TI 方案不适用的原因；
- 改用其他厂商官方资料前应取得用户确认；
- 仍须保存原厂链接、版本和设计边界。

## 从参考图到项目原理图

实际原理图不能只截取或照抄典型应用图。每个设计都应保留下面的映射记录：

| 项目 | 必须记录的内容 |
| --- | --- |
| 参考来源 | TI 器件型号、数据手册编号/版本、参考图章节或图号 |
| 工作条件 | 输入/输出电压、电流、负载、频率、温度和启动条件 |
| 参数计算 | 电阻、电容、电感、补偿、采样、保护阈值及额定值依据 |
| 项目改动 | 与参考图不同的连接或数值，以及修改理由 |
| 保护与接口 | ESD、过压、反接、浪涌、限流、电平兼容和失效状态 |
| 布局要求 | 去耦、回流路径、模拟/开关节点、散热、爬电距离和接地 |
| 验证证据 | 计算、仿真、ERC、DRC、台架测试、温升及异常工况结果 |

ERC/DRC 只能证明规则层面的连接和布局检查通过，不能替代电气计算、器件额定值检查或真机测试。

## 当前 ADS1115 + NTC100K 的特别注意事项

- ADS1115 是已明确的 TI 器件，设计依据是 TI 官方 ADS1115 数据手册。
- `NTC100K B3950` 只描述 R25 和 Beta 参数，不是完整制造商料号；不同 NTC 的 Beta 端点、公差、R/T 曲线、自热和封装可能不同。
- TI 没有可直接替代所有 NTC100K B3950 的通用离散热敏电阻数据手册，因此不能把 TI 规则错误延伸到未知 NTC。
- 在选定实际 NTC 原厂料号前，当前分压图是可编译示例接线，不是已批准的生产原理图。
- 100 kΩ/100 kΩ 分压器在 25 °C 的 Thevenin 阻抗约为 50 kΩ；ADS1115 输入负载可能引入系统偏差。高精度版本需要低偏置缓冲器、精密固定电阻、具体料号 R/T 表或 Steinhart–Hart 系数，以及整机两点或三点校准。
- 16 位分辨率不代表 16 位温度准确度。当前方案应先以全温区约 ±0.5 °C 为优化目标；未完成实物误差预算和校准前，不承诺该指标。
