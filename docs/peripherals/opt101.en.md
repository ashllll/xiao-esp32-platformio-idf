# OPT101 light sensor

OPT101 integrates a photodiode and transimpedance amplifier and produces an analog voltage related to incident optical power. Without spectral, geometric, gain, and reference calibration, its output must not be labeled lux.

## Hardware checks

- Identify the exact OPT101 module schematic and supply arrangement.
- Ensure the maximum output remains within the selected XIAO ADC input range under every expected light condition.
- Check output loading, source impedance, filtering, grounding, and decoupling.
- Keep digital and radio noise away from the analog path.

A divider can protect range but changes gain and source impedance. Record its tolerance and include it in calibration.

## Software and calibration

Use the ESP-IDF ADC calibration facilities, average only with a documented bandwidth, and retain raw ADC code plus calibrated voltage. Converting to lux requires a defined optical geometry, spectral response, and comparison against a traceable reference over the intended range.

## Acceptance

Test darkness/offset, several stable light levels, saturation, repeatability, temperature, and radio-active noise on the assembled product.
